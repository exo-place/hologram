# docs/guide/ — Summary

Step-by-step guides for users setting up and using Hologram.

## Files

- `index.md` — Guide landing page / overview
- `concepts.md` — Core concepts: entities, facts, bindings, scope resolution
- `channel-setup.md` — Binding entities to Discord channels and servers
- `personas.md` — User persona bindings (speak as an entity)
- `permissions.md` — Access control: `$view`, `$edit`, `$use`, role support, blacklists
- `multi-character.md` — Running multiple entities in one channel
- `transformations.md` — Using `$strip`, `$replace`, and output transformation directives
- `editor-setup.md` — VS Code syntax highlighting setup for `.holo` and `.njk` files
- `sillytavern.md` — Migration guide from SillyTavern character cards
